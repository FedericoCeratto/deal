
## Markers

| DEAL045 | -- `write`      | write file                |
| DEAL046 | -- `stdout`     | `sys.stdout` and `print`  |
| DEAL047 | -- `stderr`     | `sys.stderr`              |
| DEAL048 | -- `network`    | network communications, `socket` |
